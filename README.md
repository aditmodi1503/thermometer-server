# Thermometer-Server-Side-on-ExpressJs

Follow the follwoing steps to setup the server side.

  1. git clone https://github.com/aditmodi1503/thermometer-server.git
  2. cd thermometer-server
  3. npm install
  4. npm install -g nodemon
  5. Start mongo server by typing mongod in terminal (MongoDB should be installed on the system)
  6. npm start

# Configurations
1. MongoDB URL: mongodb://127.0.0.1/thermometer
2. Server port: 3001

# Conclusion
Server should be running locally on port 3001 by default and the app should run smoothly
